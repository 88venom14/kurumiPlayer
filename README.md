# KurumiPlayer

Мобильное приложение для управления вашей музыкальной библиотекой, плеера и плейлистов.

## Используемые технологии

- **React Native** — кроссплатформенная разработка
- **Expo** — управление нативными модулями
- **TypeScript** — типизация и безопасность
- **Supabase** — база данных, аутентификация, облачное хранилище
- **Zustand** — управление состоянием
- **React Navigation** — навигация между экранами

## Как запустить

### Требования
- Node.js 18+
- Expo CLI: `npm install -g expo-cli`
- Expo Go

### Установка зависимостей
```bash
cd kurumiPlayer
npm install --legacy-peer-deps
```

### Запуск приложения
```bash
npx expo start --clear
```

### Шаг 3: Подключение телефона
- Откройте **Expo Go** на своём мобильном устройстве
- Отсканируйте **QR-код** из терминала
- Приложение загрузится автоматически

## Как использовать

### 1. **Авторизация**
Зарегистрируйтесь или войдите в приложение через экран Auth Screen.

### 2. **Загрузка музыки**
- Перейдите на вкладку **Библиотека**
- Нажмите на кнопку `+` внизу справа
- Выберите аудиофайл и (опционально) обложку альбома
- Трек загрузится в облако

### 3. **Воспроизведение**
- Нажмите на трек в библиотеке
- На экране **Плеера** используйте кнопки управления
- Включите **перемешивание** или **повтор одного трека**
- Используйте прогресс-бар для перемотки

### 4. **Плейлисты**
- Перейдите на вкладку **Плейлисты**
- Нажмите `+` для создания нового плейлиста
- Откройте плейлист и нажмите `+` чтобы добавить треки
- Нажмите на иконку **поделиться** для обмена кодом плейлиста
- Используйте иконку **ссылки** для импорта чужого плейлиста

### 5. **Профиль**
- На вкладке **Профиль** установите аватар и имя
- Смотрите статистику: количество треков и время прослушивания

### 6. **Удаление**
- В **Библиотеке**: нажмите на иконку корзины рядом с треком
- На **Плейлисте**: нажмите на иконку минуса рядом с треком
